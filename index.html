The Basics of the Web

The web

 The internet is the world's largest computer network, which is comprised of many HTML documents connected together with hyper links.

The web was spun in the early 1990's stemming from the work of Tim Berners-Lee.


One can interact with over 30 billion pages by running a program called a browser on their computer. From there, the browser uses HTTP protocol to connect to servers, which are the computers that host the files that make up the web.

HTML

Hyper Text Markup Language is the Heart of the web.

HTML is made up of:
  • Text content
  • Markup
  • Refernces to other documents
  • Links to other pages

HTML markup is made up of many tags. You use tags to form elements. Tags allow you to do many things such as add images, video, white space, among many other things. Tags like br, or break, are void tags and do not require a closing tag.
